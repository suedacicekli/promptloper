# Cloudflare Dashboard Build Configuration

Use these settings in Cloudflare Dashboard → Pages → promptloper → Settings → Builds & deployments:

## Build Configuration

**Framework preset:** Next.js (Static HTML Export)

**Build command:**
```
npx @cloudflare/next-on-pages
```

**Build output directory:**
```
.vercel/output/static
```

**Root directory:** (leave empty)

**Environment variables:**
- `NODE_VERSION`: `20`

## Deploy Command

**IMPORTANT:** Leave the "Deploy command" field **EMPTY**

Cloudflare Pages automatically deploys the build output after the build completes.
Do NOT use `wrangler deploy` or any custom deploy command.

## Notes

- GitHub integration handles automatic deployments
- Each push to main branch triggers a new build
- Build succeeds → Cloudflare deploys automatically
- No manual deploy command needed
